# Kihson 
Custom JSON processing library in C programming language.


## Goals:
- Make it fast (minimal allocations + other optimizations)
- Easy to use (programmer friendly API)
- Portable (make it work with no-std?)


## Compilation:
Dependencies:
    `meson`
    
Linux:
```
$ mkdir build/
$ meson setup build/
$ cd build/
$ meson compile
```
